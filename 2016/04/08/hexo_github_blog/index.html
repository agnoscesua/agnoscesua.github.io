Hexo+GitHub 搭建自己的博客

发表于 2016-04-08

作为一名CS专业的学生，感觉需要一个个人的博客来记录学习的过程和心得了，所以第一步是搭建自己的博客。Hexo是一款基于Node.js的静态博客框架，这篇教程是针对Windows用户的。

搭建博客的前期环境准备

安装 Node.js 用于生成静态页面

安装 Git 将博客部署到GitHub上

申请 GitHub 账号

关于SSH的配置这里就不多讲了，可以看我的下一篇Git教程，这里主要讲建立和部署博客。

安装Hexo

上面步骤都完成后，开始安装Hexo，在桌面上点击右键，选择Git Bash Here，输入如下命令安装

$ npm install -g hexo

安装完成后自己选择目录建立文件夹hexo(如G:\hexo)，将来博客的配置文件都在这里，在你新建的文件夹里点击右键，选择Git Bash Here，输入如下命令初始化hexo

$ hexo init

接着输入

$ npm install

安装依赖包

至此全部安装工作已经完成，hexo文件夹就是你博客的根目录

接下来生成静态页面

$ hexo generate(hexo g也可以)

启动本地服务

$ hexo server

预览自己的博客，打开浏览器，输入localhost:4000,就可以看到自己的博客啦

配置GitHub

新建Repository

这里要注意，仓库名字必须是【你的GitHub用户名.github.io】(如zhangsan.github.io)，然后复制这个仓库的HTTP地址

编辑hexo文件中的_config.yml文件，修改其中的deploy

type：github

repository：刚刚复制的仓库HTTP地址

branch：master

最后一步，输入如下指令

$ hexo generate
$ hexo deploy

OK,现在博客已经完全搭建起来了，在浏览器中输入 yourgithub用户名.github.io(如zhangsan.github.io)

最后，每次修改完hexo本地文件后，要执行

$ hexo generate
$ hexo deploy

才能更新，当然也可以先在本地预览，没问题之后再执行第二条语句。
